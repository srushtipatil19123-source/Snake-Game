# Snake Game (Console Version)

---

## Description
A simple Python console-based Snake Game where the player controls a snake using the **W, A, S, and D** keys. The snake grows longer by eating food, and the game ends if the snake hits the wall or itself.

---

## Features
- Console-based gameplay
- Snake movement using W, A, S, D keys
- Random food generation
- Snake grows after eating food
- Wall and self-collision detection
- Score tracking
